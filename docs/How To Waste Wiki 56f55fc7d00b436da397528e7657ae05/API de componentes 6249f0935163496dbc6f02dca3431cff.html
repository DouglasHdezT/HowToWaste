API de componentes

La aplicación móvil esta desarrollada con React Navite, tecnología enfocada a componentes funcionales con un estado reactivo.

App

Componente principal en el cual se cargan las fuentes a utilizar, componentes de la aplicación, pantalla de inicio.

States:

dataLoaded: determina si ya se cargaron los recursos necesarios. | Boolean | Default: false

MainNavStack

Stack de componentes.

MainTabHome

Componente que contiene el menú inferior

RecyclerPlaceScreen

States:

mapRegion: ubicación del centro de reciclaje | Object | Default: 

{
		latitude: 0,
		longitude: 0,
		latitudeDelta:  0.005,
		longitudeDelta: 0.005,
	}

MapTab

Componente encargado de mostrar el mapa y los centros de reciclaje cercanos.

States:

region: Ubicación visual en el mapa | Object | Default: undefined.

position: localización del usuario por GPS | Object | Default: undefined.

circleRadius: radio del buffer circular | Number | Default: 1000

directionsData: data procesada de las direcciones de centros de reciclaje | Array | Default: []

selectedPlace: lugar de reciclaje seleccionado | Object | undefined;

Methods:

fetchDirections: Consulta a la api por las direcciones de los centros de reciclaje.

navigateToRP: Navega a la Hoja de información de un centro de reciclaje específico.

lineUpDirections: Transforma la respuesta de direcciones de la API en un arreglo lineal procesable por la app

CameraTab

Componente encargado de escanear imágenes.

States:

hasPermission: determina si el dispositivo tiene permisos de utilizar la camara | Boolean | undefined.

width: Ancho de la vista de cámara | Number | Default: 0 .

height: Alto de la vista de cámara | Number | Default: 0.

ModedText

Componente para mostrar texto.

Props:

big: tamaño del texto | number | Default: null.

white: Determina si el texto es color blanco | Boolean | Default: false.

center: Determina si el texto estará centrado horizontalmente | Boolean | Default: false.

centerV: Determina si el texto estara centrado verticalmente | Boolean | Default: false.

tab: Determina si el texto tendra un marginStart de 16 | Boolean | Default: false.

CameraLayout

Componente donde muestra controles de la cámara.

Props:

camera: referencia de un objeto camara. | Ref | Default: undefined.

flipCameraAction: acción de voltear camara. | Function | Default: undefined.

States:

loading: bandera que determina si se muestra el progreso de carga de la predicción | Boolean | Default: false.

complete: bandera que determina si muestra el contenido de la predicción | Boolean | Default: false.

height: tamaño del layout inferiror | Height units | Default: "15% .

modal: bandera que determina la visualizacion del modal para entrenar | Boolean | Default: false.

photo: metadata de la foto tomada | Object | Default: null.

materials: arreglo de los materiales obtenidos de la API | Array | Default : [ ].

material: material seleccionado en el modal | String | Default: null .

materiaID: material id seleccionado en el modal | String | Default: null .

objects: arreglo de los objetos | Array | Default : [ ]

object: objeto seleccionado | String | Default: null.

prediction: informacion de predicción | Object | Default: 

{material:"",object:"",precision:0}

Methods:

fetchMaterials: obtención de datos de la API .

retry: re establecimiento de estados: complete, height.

colorMaterial (material) : Obtiene un texto y devuelve un color en formato hexadecimal.

takePicture: método asíncrono que toma captura de lo que se esta visualizando la camara y enviar foto a la API y recibir la predicción

sendPrediction: enviar información sobre objetos no identificados por la API
