Insertar un nuevo objeto a un material existente.

Method

"method": "POST"

Request Body

"_id": "Material id",
"item": "New item"

Example

Request

URL: /api/material/item

BODY: {
	"_id": "5f03c85d81b92b00d3cf11e4",
	"item": Vaso de Plástico
}

Response

{
  "message": "Item inserted"
}

Clasificador de objetos

Aprender un ejemplo de un objeto

Method

"method": "POST"

Request Body

"materialID": "Material id",
"item": "item name",
"image": "item image"

Example

Request

URL: /api/classifier/saveModel

BODY: {
	"materialID": "5f03c85d81b92b00d3cf11e4",
	"item": Vaso de Plástico,
	"image": Multipart Form data
}

Response

{
  "message": "Model saved Successfully"
}

Clasificar un objeto de acuerdo a la base de conocimiento

Method

"method": "POST"

Request Body

"image": "item image"

Example

Request

URL: /api/classifier/classify

BODY: {
	"image": Multipart Form data
}

Response

{
  "type": "Caja de cartón",
  "material": "Papel",
  "probability": 1
}

Sobre rutas desactivadas

Lugares de reciclaje

  • /api/recyclerPlace/dumpData → Ingresa todos los centros de reciclaje a la Base de Datos.

Materiales reciclables

  • /api/material/dumpData → Ingresa todos los materiales reciclables a la Base de Datos.
  • /api/material/ (Method: PUT) → Actualiza los valores de un material.

Clasificador de objetos

  • /api/classifier/getAll → Obtiene todos los tensores almacenados como base de conocimiento
  • /api/classifier/dropAll → Borra todos los tensores almacenados como base de conocimiento
